# Tic Tac Toe

A simple and interactive **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**. Challenge a friend in this classic two-player game directly in your browser.

## Features

* Two-player gameplay
* Real-time win and draw detection
* Restart game functionality
* Responsive and user-friendly interface
* Fast and lightweight implementation using vanilla JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Run the Project

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Open `index.html` in your browser.

## Game Rules

* The game is played between two players: **X** and **O**.
* Players take turns marking empty cells on the board.
* The first player to get three marks in a row, column, or diagonal wins.
* If all cells are filled and no player wins, the game ends in a draw.
